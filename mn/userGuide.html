1. товч танилцуулга

Монгол NVDA дэлгэц уншигч программд тавтай морил!

NonVisual Desktop Access (NVDA) нь үнэгүй, нээлттэй эхтэй Microsoft компаны windows үйлдлийн систем дээр ажилладаг программ юм. Энэхүү дэлгэц уншигч программ нь брайлль бичгийн төхөөрөмжүүд болон ярьдаг холлойны тусламжтайгаар харааны бэрхшээлтэй иргэдэд энгийн хүн шиг компьютер ашиглах бүрэн боломжийг олгодог. NVDA нь NV Access болон хэрэглэгч, хөгжүүлэгчдийн оролцоотойгоор бүтсэн программ юм.

1.1. Ерөнхий шинжүүд

Монгол NVDA дэлгэц уншигч программ нь харахгүй хүмүүсийг windows үйлдлийн системтэй ажилладаг төхөөрөмжтэй болон нэмэлт программуудтай харьцахад гол тулгуур болж өгдөг.

Гол шинжүүдээс дурдвал:

  • интэрнэт хөтөч программууд, цахим шуудангийн аппууд болон хэрэглэгддэг оффисын программ хангамжууд зэрэг түгээмэл хэрэглэгддэг программуудыг дэмжиж ажилладаг.
  • Өөртөө суусан ярьдаг хоолойтой бөгөөд 80 гаруй хэл дэмждэг
  • Цахим бичвэрийн мэдээллүүд болох үсгийн хэмжээ, фонтын нэр, фонтын хэлбэр зэргийг зарлах боломжтой.
  • сум очсон хэсгийн бичвэрийг аутоматаар унших ба хулганы байршлыг тодорхойлох дуун хонхтой.
  • Олон брайль дэлгэцүүдийг дэмжиж ажиллах ба брайль бичвэрийн гарыг ашиглан бичвэр бичих бүрэн боломжтой
  • флаш диск эсвэл аливаа USB оролттой хадгалалтын төхөөрмжөөс аливаа систем дээр суулгах шаардлагагүйгээр шууд ажиллах
  • Амархан ашиглажж болох ба өөртөө суусан хоолойны тусламжтайгаар программыг суулгахад гадны туслалцаа шаардлагагүй
  • 60 орчим хэл рүү орчуулагдсан бөгөөд 2018 оны NVDA 18.2 хувилбараас эхлэн Монгол хэлээр орчуулагдан гарсан билээ.
  • windows үйлдлийн системийн хувилбарууд болох 32 бит ба 64 битүүдийг дэмждэг.
  • Компьютер амарсан үед эсвэл ондоо хамгаалалтын дэлгэцтэй харьцах тохиолдолд уншдаг давуу талтай
  • Мэдрэгчтэй дэлгэцтэй төхөөрөмж дээр ажиллаж байх тохиолдолд бүхий л коммандуудыг зарладаг
  • Microsoft Active Accessibility, Java Access Bridge, IAccessible2 and UI Automation зэрэг хүртээмжтэй программын интерфэесүүдтэй бүрэн зохицож ажилладаг
  • windows үйлдлийн системдэх командын тавцан болон кодочлолын талбаруудыг дэмжин унших зэрэг багтана.

1.3. Ярьдаг программын хоолойнууд

NVDA дэлгэц уншигч программ нь зөвхөн цэс интерфэсээ ондоо хэл рүү орчуулсанаас гадна олон хэлийн хоолойг дэмжиж ажилладаг. Тухайн хэлний хоолой хийгдсэн тохиолдолд бүхий текстийг уг хоолойгоор унших юм.

NVDA дэлгэц уншигч программ нь eSpeak NG буюу и-яриа гэсэн үнэгүй нээлттэй эхтэй хоолойг гол хоолойгоо болгон ашигладаг..

1.4. Брайль

Брайль дэлгэц хэрэглэдэг хүмүүст NVDA дэлгэц уншигч программ нь мэдээллийг брайль дэлгэцээр унших сонголтийг олгодог. Брайлийн нэг хоёр дугаар хувилбаруудаар бичвэр бичих төхөөрөмж ашиглан бичих боломжийг бас бүрдүүлдэг. NVDA дэлгэц уншигчтай ажилладаг брайль дэлгэцний тухай мэдээллийг Дэмждэг брайль дулгэцнүүдийн тухай хэсгээс авна уу. NVDA дэлгэц уншигч нь брайлийн бүхий л хувилбаруудыг олон хэл дээр гаргаж болдог. Үүнд: брайль нэг, брайль хоёр болон компьютер брайль гэх мэтт.

NVDA программыг суулгаснаар та нэмэлт үйлдлүүдийг ашиглах боломжтой болох ба үүнд: NVDA программ компьютер асах тохиолдолд автоматаар ажиллах, windows системийн user log on буюу бүрэн асаагүй тохиолдолд ажиллах, мөн дэлгэц дээр NVDA программын icon-ийг гаргах гэх мэт. Эдгээр нь түр хуулбар болон зөөврийн хувилбаруудадтай ашиглах боломжгүй үйлдлүүд юм.. Мөн программын суусан хувилбар өөрийн зөөврийн хувилбарыг ямарч тохиолдолд гаргах боломжтойгоороо давуу талтай байдаг.

Харин та NVDA программыг флаш диск эсвэл СД кард зэрэг медиа файл зөөвөрлөгчинд хадгалж үүнийгээ ямарч компьютерт залган ажиллуулахыг хүсвэл NVDA проргаммын зөөврийн хувилбарыг сонгох нь зүйтэй. Зөөврийн хувилбараас ямарч компьютерт NVDA программыг бүрэн суулгах боломжтой учир, олон хүмүүс энэхүү хувилбарыг түгээмэл ашигладаг. Харин та nvda программыг зөвхөн унших зориулалттай агуулах дискүүд болох CD-DVD дээр зөөврийн хувилбарыг оруулан ашиглахыг хүсэж байвал бид таныг суулгагчийг л оруулахыг зөвлөж байна. Мөн хэрэглэгч стерио дуугаралттай чихэвч эсвэл чанга яригч хэрэглэж байгаа тохиолдолд хулганы сум дэлгэцний зүүн тал руу хөдлөх тусам уг чимээ маань даган зүүн талын чихэнд илүү тод сонсогдож, харин баруун тал руу нүүхэд баруун талд дуугарах гэх мэтээр дэлгэцний аль хэсэгт сум байгааг бүрэн илэрхийлдэг.

Эдгээр хулганы нэмэлт тохиргоонууд нь NVDA программын үндсэн тохиргоонд тохируулагдаагүй байдаг. Хэрэв та уг нэмэлт шийдлийг ашиглахыг хүсвэл NVDA программын сонголтууд цэсний дотор байрлах тохиргоо цэснээс хулганы тохиргоо хэсгээс сонгон тааруулж болох юм.

6. Хөтөч горим

Нарийн хийцтэй баримт бичиг болон бичвэрүүд болох вэб хуудас зэрэг нь энэ горимыг ашиглан уншигддаг байна. Энэ горимтой бүрэн ажиллах бичвэрийг унших программууд нь Mozilla Firefox, Microsoft Internet Explorer, Mozilla Thunderbird, HTML messages in Microsoft Outlook, Google Chrome, Adobe Reader and Adobe Flash зэрэг юм. Хөтөч горим нь мөн Microsoft компаны оффисын программуудтай бүрэн ажилладаг.

Хөтөч горимын үндсэн зорилго болон давуу тал нь аливаа бичвэрийг гарын үсгэн болон бусад товчлуурны командуудыг ашиглан хялбар аргаар унших боломжийг бүрдүүлдэгт оршино. NVDA программын бичвэрийн сум систем каретийн командууд энэхүү горимтой бүрэн ажилладаг; Жишээлбэл, бүгдийг унших, бичвэрийн мэдээллийг хэлэх, хүснэгттэй харилцах командууд гэх мэт. Бичвэрийн талаар нэмэлт мэдээллүүд болох холбоос, гарчиг эсвэл товчлуур зэрэг нь NVDA программ тухайн бичвэрийг уншихад давхар уншигдах зарчмаар ажиллладаг байна.

Зарим тохиолдолд та энэ төрлийн бичиг баримтуудтай ажиллахдаа нэмэлт командуудыг ашиглах шаардлага гардаг. Жишээ нь, та бичвэр бичих хэсэг дээр бүхий л л бичигдэх үсэг/тэмдэгтийг систем руу бичвэр хэлбэрээр дамжуулах, үгүй бол хөтөч горим руу шилжүүлэн засварлаж буй бичвэртэйгээ хялбар харилцах хэрэгтэй тохиолдлуудад хөтөч горимыг нээж/хаах шаардлагууд гардаг билээ. Эдгээр тохиолдлуудад хөтөч горим болон төвлөрсөн горимуудын хооронд шилжүүлэн ажиллах бүрэн боломжтой. Ихэнх тохиолдолд NVDA программ нь дээр дурдсан программуудын бичвэртэй харилцахдаа автоматаар хөтөч горим руу шилжүүлдэг. Үүний тод жишээ бол интернэт ордог хөтөч программууд юм. Хөтөч горимын үед хэрэв та аливаа бичвэрийн эсвэл бичиг оруулах зориулалттай обект дээр enter болон space товчийг дөрөн төвлөрсөн горим руу шилжүүлж бүхий л оруулж буй үсгүүд болон тэмдэгтүүд команд биш систем руу дамжуул гэсэн утгатай үйлдлийг гүйцэтгэх шаардлагатай юм. Харин tab товчлуурыг дарах эсвэл escape товвчлуурыг дарах, зарим тохиолдолд тухайн обектийг дахин дарж гэмээн буцаж хөтөч горим руу шилжүүлнэ. Үүнээс гадна та хүчээр хөтөч горим болон төвлөрсөн горимын хооронд солих боломжтой ба, аль нэгийг нь сонгосон тохиолдолд буцааж сольж байж л тус тусын үүргүүдийг гүйцэтгэх юм.

  • u: зочлогдоогүй холбоос

6.2. Элементүүдийн жагсаалт

Элементүүдийн жагсаалт нь тухайн бичвэрийн олон арван элементүүдийг төрөлжүүлэн шилжүүлэх боломжийг олгодог хөтөч горимын нэг гайхалтай үзүүлэлт команд юм. Жишээ нь, Интернэтийн орчинд бичвэрийн элементүүдийг холбоос, гарчиг, жагсаалт, товчлуурнууд, чиглүүлэч болон бичвэрийн талбарууд зэрэг элементүүдийг төрөл төрлөөр нь үзүүлэх хэрэглэхэд энгийн нэг төрлийн харилцах цонх юм. Уг харилцах цонхонд байх бүлэг чагтнууд нь аль төрлийн элементийг жагсааж үзэхээ сонгоход тус болдог. Мөн тус харилцах цонхонд байх бичвэрийн талбар нь төрөлжүүлсэн элементүүдийн жагсаалтаа ялгаварлаж үзэхэд тусладаг сонголт юм. Жагсаалтан дотроосоо аль нэг очихыг хүссэн обект руугаа очих, эсвэл тухайн элементийг ажиллуулахыг хүсвэл өгөгдсөн товчлуурнууд дээр дарахад л тухайн үйлдлийг цаг алдалгүй гүйцэтгэдэг.

9. Контент танигч

Зарим мэдээллийн сайтууд, хуудаснууд болон аливаа нэг бүтээлийн зохиогч NVDA программ хангалттай таньж ажиллахгүй контентүүд бүтээлдээ оруулсан бол үүнд нэмэлтээр хэд хэдэн программ аппликэешинүүдийг ашигладаг. NVDA нь windows 10 үйлдлийн системд байх OCR буюу (үсэг танигч)-ийг дэмжиж ажилладаг бөгөөд энэ нь аливаа зурагнаас бичгэн мэдээллийг таньж хөрвүүлдэг прорграмм юм. Мөн нэмэлтээр NVDA-гийн нэмэлт программ хэлбэрээр windowsын өмнөх хувилбарууд дээр суулгаж ч болдог.

OCR (үсэг танигчийг) ашиглахад NVDA программ нь Очсон обект дээр байгаа зурагны бичвэрийг өөртөө ойлгомжтой хэлбэрээр хөрвүүлдэг. Үндсэн тохиргоогоороо бол обектоор шилжүүлэх горим нь системийн төвлөрлийн горим эсвэл нарийвчлан үзэх горимын сумыг автоматаар дагаж байдаг учраас хүссэн газраа аваачин OCR(үсэг танигч)-ийг ашиглаж болох юм. Жишээ нь, хэрэв та хөтөч горимын сумыг graphic буюу зураг дээр аваачаад хувиргахад тухайн зурагны агуулж буй бичгийг OCR маань хөрвүүлж NVDA программ унших боломжтой болгодог байна. Мөн та OCR(үсэг танигч)-ийг ашиглан бүхэл бүтэн программын цонхыг хувиргаж болох аж.

Хувиргах үйл явц дуусахад үр дүнг виртуал хөтөч горимын төрлийн бичвэр хэлбэрээр гаргадаг. Тухайн үр дүндэх бичвэрийг ажиллуулахын тулд түүн дээр enter эсвэл space товчлуур дарна. Харин escape товчлуурыг ашиглаж гарсан үр дүнг хаадаг байна.

Профайлыг дахин нэрлэх эсвэл устгах бол тус тусын товчийг дарна.

Харин профайлын цонхыг хаахдаа хаах товчийг дарж гаргана.

11.3.2. Профайл үүсгэх нь

Шинэ нэртэй товч дээр даран профайл үүсгэнэ.

Гарч ирэх бичвэрийн талбарт профайлын нэрийг өгнө. Мөн уг профайл нь хэрхэн хэрэглэгдэх талаар тохируулгыг тохируулж сонгоно. Тухайн профайлыг гараар ажиллуулах сонголтыг сонговол энэ нь гараар нээгдэх ба эсэргээрээ аливаа нэг программтай эсвэл алхам гүйцэтгэж байх үед автоматаар нээгдэх бол тухайн хэсгийг сонгон тохируулна. Профайлын нэрийг өгөөгүй тохиолдолд автоматаар ажиллуулах сонголт өөрийн нэрийг өгөх болно. Автоматаар ажиллуулах тохиргооны талаар дэлгэрэнгүй мэдээллийг дараах хэсгээс аваарай.

Эцэст нь тийм товчийг даран профайлаа үүсгэн тохиргоонуудаа тохируулж эхлэхэд бэлэн болно.

11.3.3. Гараар ажиллуулах

Профайлаа сонгосны дараа гараар ажиллуулах товчийг даран профайлыг ажиллагаанд оруулна. Автоматаар ажилладаг профайлууд энгийн үед ажиллах хэдий ч гараар тохиргооноос ажлуулж байгаа профайл тэднийг даван ажилна. Жишээ нь, одоо ажиллаж байгаа автоматаар ажилласан профайл дээр холбоосыг уншина гэсэн тохируулга хадгалагдсан байгаад сүүлд ажиллагаанд орсон гараар ажиллуулах профайл дээр холбоос уншихгүй гэсэн бол хоёрдогч профайл нь автоматаар ажиллах профайлын тохиргоог даран ажилна гэсэн үг юм. Гараар ажлуулсан профайлаа хаахыг хүсвэл үндсэн цэс дотроос тухайн профайлаа сонгон ажиллагаагүй болгох товчийг даран гаргана.

11.3.4. Автоматаар профайл ажиллуулах

Профайлдаа автоматаар ажиллуулах түгжээ хийхийн тулд профайл үүсгэхдээ гарар биш бусад сонголтуудаас сонгон идвхижүүлнэ.

Автоматаар ажиллуулах тохиргоо нь:

  • одоогийн аппликэешин: тохируулсан программ ажиллахад профайл ажилна.
  • бүгдийг хэлэх: бүгдийг унших команд орох үед профайл ажиллах.

Автоматаар ажиллуулах төрлөө сонгосны дараа аль профайлыг ажиллуулах гэдгээ сонгон идэвхижүүлнэ. Профайл хэрэглэхийг хүсэхгүй бол "(Ердийн тохируулга)" сонголтыг сонгож болно.

Хаах товчийг даран буцан профайл харилцах цонх дээрээ ирнэ.

11.3.5. Профайл засварлах

Хэрэв та профайлыг гараар ажлуулсан бол аливаа өөрчлөлтүүд тухайн профайлд хадгалагдах болно. Харин үгүй бол өөрчлөлтөнд орсон тохиргоонууд сүүлчийн ажиллаж байсан профайлд орно. Эцэст нь гараар ажиллуулах ч биш автоматаар ажиллах ч профайл биш байвал таны хийсэн өөрчлөлтүүд ердийн тохируулга буюу үндсэн тохиргоонд орох юм.

11.3.6. түгжээнүүдийг түр хаах

Заримдаа автоматаар шилжүүлдэг түгжээнүүдийг хаах шаардлага гардаг. Жишээ нь, ердийн болон гараар тохируулдаг профайлуудыг янзлах үед автоматаар ажилладаг түгжээнүүд саад болох тохиолдлууд байдаг. Үүнийг хийхдээ профайлын тохиргоо харилцах цонхны бүх түгжээг хаах чагтыг чагталсанаар хааж болох байна.

11.4. Тохируулгын файлын байршил

Зөөврийн хувилбар нь аливаа тохиргооны файл болон нэмэлт программын файлуудыг Userconfig гэсэн тухайн NVDA байрлаж байгаа байршлын хавтсанд хадгаладаг.

Харин бүрэн суусан NVDA программын хувилбарын тохиргооны файлууд нь тухайн үйлдлийн системийн хэрэглэгчийн профайл хэсэгт байрладаг. Энэ нь тухайн үйлдлийн систем дээр буй хэрэглэгч тус бүр ондоо NVDA программын тохиргоотой байж болно гэсэн үг юм. Тухайн байршил руу очихдоо хайлтын хайрцагнаас NVDA explorer гэсэн хавтас руу нэвтэрч болно.

Хэрэглэгчийн тохиргоо болон цоожтой тавцангийн тохиргоонууд нь үндсэн системийн байршил болон NVDA программын суусан байршилд байрладаг. Тиймээс уг тохируулгыг хөдөлгөж зөөх нь буруу зүйл юм. Цоожтой тавцангийн эсвэл системийн хэрэглэгчийн тохиргооны тохируулгыг өөрчлөх бол тухайн өөрчлөх янзаараа NVDA программын тохиргоогоор өөрчлөн ерөнхий тохиргоо дотор байрлах windows цоожтой үед дэлгэц уншигч программ автоматаар эхлэх сонголтыг ашиглан тохируулна.

12. Нэмэлтт хэрэгслүүд

12.1. Лог хураагч

Лог хураагч нь NVDA программын бүхий л лог буюу ассанаасаа хойших ажиллаж байгаа мэдээллийг хурааж байдаг хэрэгсэл юм. Үүнийг программын цэсний хэрэгслүүд цэснээс олж болно.

Программын логийг үзэхээс гадна эдгээрийг хуулж авж болох ба мөн дахин эхлүүлж үзэж болох юм . Эдгээр сонголтуудыг тухайн лог хураагчаас олж харж болно.

12.2. Яриа хураагч

Хараатай NVDA дэлгэц уншигч программын хөгжүүлэгчдэд NVDA программын яриа сонсогдохгүйгээр харин юу ярьж байгаа нь урсах хэлбэрээр гаргаж өгдөг хэрэгслийг яриа хураагч гэнэ.

Үүнийг нээхийн тулд NVDA программын хэрэгслүүд цэсэнд байрлах ярианы ажиглагч сонголтыг чагтлан ажиллагаанд оруулна. Харин хаахдаа чагтгүй болгож энэ хэрэгслээс гаргана.

Уг хэрэгсэл нь программ эхлэхэд асах чагтийг дотроо агуулдаг. Чагтлагтсан тохиолдолд дэлгэц уншихгч асах үед дагаж асна гэсэн үг юм.

Яриа ажиглагч нь нээлттэй үед бүх яригдаж буй зүйлийг шинэчлэн гаргаж байдаг. Харин тухайн яриа гарж байгаа хэсэгт дарахад эсвэл оруулахад түр зогсож тэдгээр яриаг хуулж авахад боломжтой болгож өгдөг.

Энэхүү хэрэгслийг гарын холбоостой болгохыг хүсвэл Гарын команд тохируулах харилцах цонх дотор тохируулна уу.

12.3. Add-ons зохицуулагч

Add-on буюу нэмэлт програмууд нь NVDA дэлгэц уншигч программд нэмэлт үйлдлүүд болон функциудыг суулгахад ашиглагддаг багц юм. Эдгээр багц нэмэлт программууд нь NVDA хөгжүүлэгч бүлгэм хүмүүсийн дунд хөгжин нэмэгдэж байдаг.

Add-on жагсаалт нь одоо суусан байгаа багц нэмэлт программуудын жагсаалтыг харуулдаг. Add-onы зохиогч, ажиллаж буй байдал болон хувилбарын тухай мэдээллүүдийг ерөнхийдөө зарладаг боловч илүү дэлгэрэнгүй мэдээллийг add-onы тухай товч дээр дарж авна. Хэрэв тухайн нэмэлт add-onы талаар тусламж файл байвалл үүнийг add-on тусламж товч дээр даран сонсоно.

Add-on татаж авахыг хүсвэл add-on татаж суулгах товч дээр дарна. Энэ товч нь add-onы хуудас болох NVDA Add-ons page-ийг нээдэг юм. Хэрэв NVDA программ суусан бол вэб дээрээс нь шууд суулгах боломжтой. Харин үгүй бол татаж хадгалаад дараа нь суулгах боломжтой байдаг байна.

Өмнө нь татсан add-on суулгахын тулд add-on суулгах товчийг дарна. Энэ нь компьютер дээрээсээ add-on байгаа газрыг зааж өгөх үйлдэл гүйцэтгэх харилцах цонхыг нээдэг. Add-on файлаа сонгоод нээх товчийг дарснаар суулгаж эхэлнэ.

Add-on суухдаа юуны өмнө таниас тухайн add-onыг суулгахдаа итгэлтэй байгаа эсэхийг лавтган асууна. Add-on н нь өмнө дурдсан шиг нэмэлт программ учир итгэлгүй эх сурвалжаас татсан багцууд вирус эсвэл таны хувийн мэдээллийг хулгайлах зорилготой программ байж мэдэх учир албан ёсны сурвалжаас татан суулгахыг санал болгож байна. Сууж дууссаныхаа дараа дэлгэц уншигч программ нь дахин эхлэх ёстой ба дахин дахин эхэлсэний дараа add-on нь бүрэн ажиллагаанд ордог.

Add-onыг устгахда устгах товчин дээр дарна. Устгасныхаа дараа мөн программыг дахин эхлүүлэн бүрэн устгах хэрэгтэй.

Add-on түр хаах бол ажиллагаагүй болгох товч дээр дарна. Харин өмнө нь хаасан add-on буцааж нээхдээ нээх товч дарна. Add-on хаах болон нээх товч дарагдахад тухайн add-onы талаар хаалттай гэсэн мэдээлэлтэй болдог. Мөн add-on нээж хаахдаа заавалчгүй NVDA программыг дахин эхлүүлж байж тухайн өөрчлөлтүүд идэвхижинэ.

Уг хэрэгсэл нь мөн хаах товчтой бөгөөд тус товчийг дарснаар тухайн харилцах цонхыг хааж болно. Харин ямар нэг өөрчлөлт оруулсан бол хаах товчийг дарахад программыг дахин эхлүүлэх тухай тийм үгүй товч гарч ирнэ.

Add-on зохицуулагчийг гарын холбоостой болгохыг хүсвэл Гарын команд тохируулах харилцах цонх дотор тохируулна уу.

14. Дэмжигдсэн брайль дэлгэцүүд

NVDA программ нь нэмэлтээр нэг юм уу түүнээс олон тооны командын мөрний сонголтуудыг зааж өгсөнөөр үндсэн тохиргооноос өөрөөр ажиллах боломжтой байдаг. эдгээр командын мөрний бичгэн тохируулгуудыг богино холбоосын properties (alt enter дарсаны дараа), run харилцах цонхноос буюу (Windows+r дараад эсвэл эхлэх цэснээс run сонголтыг олж дарсаны дараа) эсвэл windows үйлдлийн системийн командын харилцагчаас оруулж ашиглаж болдог. ТОхируулган командууд нь NVDA программын нэртэй гүйцэтгэгч файлаас үргэлж зайгаар тусгаарлагдах ёстой байдаг. Жишээ нь, NVDA программыг суулгаж байх үед дэлгэц дээр үүсдэг NVDA богино холбоос нь дотроо -r командын тохируулгыг агуулсан байдаг ба энэ нь NVDA программыг одоо ажиллаж байгаа хувилбарыг хаагаад шинээр нээнэ гэсэн утгатай захирамжийг агуулдаг. Мөн ондоо нэг хэрэгтэй команд нь --disable-addon юм. Энэ нь NVDA программд одоо ажиллаж байгаа бүхий л add-on-уудыг зогсоо гэсэн гүйцэтгэлийг оруулан бүх ажиллаж байсан нэмэлт программуудыг хаадаг байна. Энэхүү командыг ашигласнаар аливаа асуудал нь add-on-оос болж үүсэж байгаа эсэхийг, эсвэл аливаа арилахгүй байгаа асуудал add-on нэмэлт программаас болж үүссэн бол тэдгээрийг засахад тун их хэрэгтэй юм.

Жишээ болгоод дараах командыг run харилцах цонх дээр оруулан одоо ажиллаж буй NVDA программыг хааж болох юм:

nvda -q

Зарим командууд нь урт болон богино гэсэн хоёр ондоо хувилбартай байдаг бол зарим нь зөвхөн уртаараа бичигддэг. Богино хувилбартай командуудыг доорх байдлаар нэгтгэн ашиглаж болно:

nvda -rm энэ команд нь NVDA программыг эхлэх хөгжмийг тоглуулахгүйгээр дахин эхлүүлэх гэсэн команд, гэх мэтээр ашиглаж болох аж.
nvda -rm --disable-addons дээрх командтай ерөнхийдөө адилхан ч нэмэлтээр add-on нэмэлт программуудыг хаах гэсэн шинэ заалт орсон байна.

Зарим командууд нь нэмэлт parameter авсанаар илүү нарийвчилсан тохируулгуудыг хийж болдог. Эдгээр нэмэлт parameter-үүд нь тухайн командын араас зай аваад богино хувилбар бол тэнцүүгийн тэмэгтэй, харин урт хувилбар бол хагас дусал хаалттайгаар бичигддэг:

nvda -l 10 NVDA программыг эхлэхдээ алдаа хянагчийн файлыг үүсгэн эхлэх
nvda --log-file=c:\nvda.log NVDA программд өөрийн log файлыг c:\nvda.log байршилд бич гэж зааж өгөх х
nvda --log-level=20 -f c:\nvda.log NVDA программыг эхлэхдээ log файлаа мэдээллийн хэлбэрээр c:\nvda.log байршилд бич гэсэн даалгаврыг өгөх гэх мэт.

15.2. Систем Wide Parameters

NVDA программ нь мөн системийн бүртгэл дотор ситем wide parameter-ийн утгуудыг өөрчлөн ондоо үйлдлүүдийг гүйцэтгэх боломжтой байдаг. Эдгээр утгууд нь системийн бүртгэл дотор дараах түлхүүрүүдийн аль нэгийг аван байрладаг:

  • 32-bit system: "HKEY_LOCAL_MACHINE\SOFTWARE\nvda"
  • 64-bit system: "HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\nvda"

Доорх утгууд нь эдгээр түлхүүрийг ашиглан системийн бүртгэлд багтана:

Нэр Төрөл боломжит утгууд тайлбар
configInLocalAppData DWORD 0 (default) to disable, 1 to enable If enabled, stores the NVDA user configuration in the local application data instead of the roaming application data
serviceDebug DWORD 0 (default) to disable, 1 to enable If enabled, disables secure mode on windows secure desktops, allowing the use of the Python console and Log viewer. Due to several major security implications, the use of this option is strongly discouraged
