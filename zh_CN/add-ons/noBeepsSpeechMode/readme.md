# NoBeepsSpeechMode #
*	 作者: Alberto Buffolino.
*	 下载 [稳定版][1]

按下NVDA + s时，此插件会在语音模式中关闭哔声.
你只能在关闭或朗读之间进行切换.

## 版本 2.0 ##
*	 现在插件管理器可以提供插件帮助.

## 版本 1.5 ##
*	 添加了脚本类别（语音）.
